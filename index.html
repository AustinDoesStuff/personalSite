Austin's Crappy Weather

Austin's Crappy Weather is a weather service deployed on Amazon Web Services. This project uses JavaScript and AngularJS to make an AJAX call to determine the user's location. After recieving the latitude and longitude coordinates, it makes another AJAX call to Open Weather Map's API to get the user's weather. The background image changes depending on what the weather is. This site is currently undergoing a massive front-end overhaul.

SpeedTest

SpeedTest is a python script that tests the speed of your ISP then tweets the results. This project uses Selenium Web Driver to automate a web browser to speedtest.net, runs the test and scrapes the results. The results are then converted to mb/s, is compared to what I am paying for and if the results are less than 80% of what I'm paying for the results are tweeted out using OAuth2 to authenticate and Tweepy to send the tweet.
